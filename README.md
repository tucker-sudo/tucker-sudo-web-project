# tucker-sudo-web-project

## Project 2 - HTML Foundation Page

For this project, I've decided to go with an informative page on 3D graphics and software. I structured it in a Wiki-style fashion.

I learned about how I can style HTML pages without the use of CSS using span styling elements. This was definitely the trickiest part for me as I hadn't done HTML-only styling too much before this project. I styled a quote block for the quote from Wikipedia for this, with a background color, font color, font size, and padding.
